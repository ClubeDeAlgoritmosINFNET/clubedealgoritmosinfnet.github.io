Sobre Nós

É um grupo dedicado para os membros treinarem lógica de desenvolvimento de algoritmos e manipulação de estrutura de dados, com foco em competições internas e externas.

Linguagens

Baseando-se nos campeonatos que ocorrem pelo mundo serão utilizadas para a resolução dos problemas as linguagens C/C++, Python 2/3 e Java como padrão, porém, o membro pode escolher a linguagem que se sinta confortável.

Treinamentos

Os treinamentos serão online e no espaço fornecido pela a universidade.

Ganhos para o membro

Inúmeras empresas internacionais como a Google, Amazon, Intel, Facebook e muitas outras, dão extrema importância ao conhecimento de lógica e estrutura de dados, bem como o desenvolvimento de algoritmos para a resolução de diversos problemas.

Além de você saber criar soluções fora da caixa e um maior entedimento de como um algoritmo funciona, você pode ter no seu curriculo acadêmico uma atividade extra.

Resoluções e Material para estudo no Github

Além de ter resoluções de alguns dos problemas passados para treinamento, teremos um repositório dedicado para compartilhar links para estudo.
